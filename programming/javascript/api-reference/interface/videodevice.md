# VideoDevice

The `VideoDevice` interface specifies a camera.

```ts
interface VideoDevice {
  /** The unique identifier for the camera. */
  deviceId: string;
  /** The label or name of the camera. */
  label: string;
}
```

## deviceId

The unique identifier for the camera.

## label

The label or name of the camera.
